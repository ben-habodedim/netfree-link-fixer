# netfree-link-fixer
תוסף Chrome שמתקן אוטומטית קישורים לשיתוף קבצים שנשברו בעזרת כוכביות (*).

## מה התוסף עושה

התוסף מזהה ומתקן אוטומטית קישורים שנכתבו בצורה שבורה כדי לעקוף חסימות.

## יכולות

תיקון קישורים עם תווים שחוזרים על עצמם יותר מ־3 פעמים  
תמיכה בכל סוגי הקישורים של Google Drive  
זיהוי רחב של קישורים גם אם שובשו בכוונה  
תמיכה בקישורי MagiCode  
ניקוי תווים מיותרים מתוך קישורים


## התקנה (Chrome / Edge / Brave)

1. הורד את הפרויקט כ-ZIP
2. חלץ את הקבצים
3. פתח בדפדפן:

chrome://extensions

4. הפעל **Developer Mode**
5. לחץ **Load unpacked**
6. בחר את תיקיית התוסף

## תמיכה בדפדפנים

Google Chrome  
Microsoft Edge  
Brave  
Mozilla Firefox (עם התאמות קטנות)

## מטרת הפרויקט

לעזור למשתמשים בקהילות מסוננות לפתוח קישורים שנכתבו בצורה שבורה כדי למנוע זיהוי אוטומטי.

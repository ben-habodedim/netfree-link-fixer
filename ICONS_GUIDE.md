# הוראות ליצירת אייקונים

לפני שתוכל לטעון את התוסף, **חייבים** ליצור 3 קבצי אייקונים:

- `icon16.png` (16x16 פיקסלים)
- `icon48.png` (48x48 פיקסלים)
- `icon128.png` (128x128 פיקסלים)

## דרך מהירה - באמצעות אתר מקוון

### 1. גש לאתר ico converter

אתרים מומלצים:
- https://www.favicon-generator.org/
- https://favicon.io/favicon-converter/
- https://www.canva.com/

### 2. צור אייקון פשוט

העלה תמונה או צור אייקון חדש עם:
- רקע כחול/ירוק
- סימן 🔗 (שרשרת/קישור)
- או האות **N** (נטפרי)

### 3. ייצא בגדלים הנדרשים

רוב האתרים יאפשרו לך להוריד את האייקון בכל הגדלים.

### 4. שנה שמות ושמור

- שנה את השמות ל: `icon16.png`, `icon48.png`, `icon128.png`
- שמור את הקבצים בתיקייה:
  ```
  c:\Users\User\Desktop\netfree-link-fixer\
  ```

---

## דרך ידנית - Paint או כל תוכנת עריכה

### 1. פתח Paint / Photoshop / GIMP

### 2. צור תמונה חדשה

- גודל: 128x128 פיקסלים
- רקע: כחול (#4285F4) או ירוק (#34A853)

### 3. הוסף טקסט או צורה

- טקסט: **N** או **🔗**
- צבע: לבן
- יישור: מרכז

### 4. שמור 3 גרסאות

1. שמור בגודל 128x128 בשם: `icon128.png`
2. שנה גודל ל-48x48 ושמור בשם: `icon48.png`
3. שנה גודל ל-16x16 ושמור בשם: `icon16.png`

## פתרון זמני מהיר

אם אתה רק רוצה לבדוק שהתוסף עובד, אפשר להעתיק כל אייקון פשוט 3 פעמים ולשנות את השמות.

**לדוגמה:**
```
copy any_image.png icon16.png
copy any_image.png icon48.png
copy any_image.png icon128.png
```
